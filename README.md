## Requirements

- Neovim
- A C compiler in your path and libstdc++ installed (gcc, g++) for TreeSitter

## Removing

To completely remove any footprints of the config:

```zsh
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.cache/nvim
```
